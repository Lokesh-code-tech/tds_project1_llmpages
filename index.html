Asset Playground

Static assets via a pure HTML/JS site (no back-end)

This page lets you download and preview a set of files that would normally live in a GitHub Pages repository. All content is generated client-side and is stored in your browser via localStorage to demonstrate persistence. Use the file panels below to preview or download each asset. You can also attach a CSV or an image to see client-side processing in action.
